Class Mapsicle

Description
Description | Vars (details) | Methods (details)

A simple SQL mapper for PHP. You can define SQL statements and map them to an id in a configuration file called 'mapsicle.xml' (or name it whatever you want, and pass it in to MapsicleFactory::createMapsicle).

For example, suppose you had a simple query:

  1.  select * from customers where customer_name = ?

Simply define this query in your mapsicle.cfg.xml as:

  1.  <select id="Customers.byName">
  2.         select cust_name         as name,
  3.             cust_addr            as address,
  4.             cust_city            as city,
  5.             cust_state           as state,
  6.             cust_zip             as zip,
  7.             cust_email           as email
  8.         from customers where customer_name = #name#
  9.  </select>

In this case, #name# is a named parameter; you will call one of Mapsicle's queryFor methods and pass in both the id and parameters. For example:

  1.  $mapsicle =& MapsicleFactory::buildMapsicle();
  2.  if (PEAR::isError($mapsicle)) {
  3.         die($mapsicle->getMessage());
  4.  }
  5.  $params = array('name' => 'John Doe');
  6.  $customer =& $mapsicle->queryForObject('Customer.byName', $params);
  7.  print $customer->email;
