		getMapping (line 238)
	
 
	
	

Create a MapsicleMapping object from a parsed XML element and the  sql statement type.  The XML element should be an array with keys '@'  and '#' if parsed by the PEAR_Config class.


i.e. The array should look like this:  
  1.  array('@' => array('id' => 'MapID', 'delimiter' => 'MapDelimiter'),
    2. 

  2.     '#' => 'select * from customer')
    3. 

The type is either 'select', 'update', 'insert', or 'delete'.

	
    
				
  • return: A MapsicleMapping object or a PEAR_Error if the element  wasn't defined correctly.
    •
