Mapsicle - SQL maps for PHP!

Mapsicle was inspired by iBATIS Data Mapper (http://ibatis.apache.org), but is not intended to be an exact port. Mapsicle has less features than iBATIS, but hopefully you will still find it useful.

PHP versions 4 and 5
